# Security policy

## Supported versions

| Version | Supported |
| --- | --- |
| 1.0.x | Yes |
| Earlier releases | No |

## Private vulnerability reporting

Do not disclose a suspected vulnerability in a public Issue or discussion. Use GitHub's
[private vulnerability reporting form](https://github.com/Hitsuki-Ban/Splatorium/security/advisories/new).

The security owner is [@Hitsuki-Ban](https://github.com/Hitsuki-Ban). Reports receive an
acknowledgement within 3 business days and an initial triage decision within 7 business days.
Confirmed issues are coordinated privately until a fix and disclosure plan are ready.

Include the affected version, environment, impact, reproduction steps, and any proof-of-concept
material that is safe to share privately. Do not include credentials, personal data, model weights,
or other third-party confidential material.
