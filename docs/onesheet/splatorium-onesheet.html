Splatorium 画像 1 枚から 3D モデルを作り、集めて並べられるローカルアプリ
原画 1 枚
→
3D モデル
1
画像を選んで、生成する 画像を選び、モデルの細かさを決めて開始。作業状況は同じ画面で確認できます。
2
できたモデルを倉庫で探す 完成したモデルは自動で収蔵。サムネイルを見ながら、名前の変更や検索もできます。
3
好きな角度から 3D を見る モデルを回転・拡大して原画と見比べます。気に入ったら、そのままシーンへ追加。
4
シーンに並べて、保存する 移動・回転・拡縮で位置を整え、複数のモデルをひとつの場面に。保存後もいつでも続きから編集できます。

必要なもの

  • Windows PC と WebGL2 対応ブラウザ
  • app/node/README.txt に記載された Node.js
  • TripoSplat 対応の ComfyUI
  • 指定のモデルファイル

はじめ方

GitHub Releases から Portable ZIP を取得して展開します。Node.js と ComfyUI を所定の場所に配置し、モデルを準備してから run.bat を実行してください。

初回設定、モデルの取得、基本操作、バックアップ方法は、リポジトリ内の日本語ユーザーマニュアルで確認できます。

動作環境

  • Windows + GPU（NVIDIA 12 GB 級推奨）
  • Intel GPU は対応する ComfyUI と PyTorch XPU 環境が必要
  • 閲覧・シーン編集は生成より少ない GPU メモリを使用
